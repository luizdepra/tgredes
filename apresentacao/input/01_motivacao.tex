\section{Motivação}

\begin{frame}
	\frametitle{Porque comparar o MySQL com árvores multidimensionais?}
	\begin{itemize}
		\item Aplicações distribuídas possuem requisitos específicos de desempenho, e para isso necessitam do gerenciamento da rede que oferece o serviço.
		\item A gerência de redes consiste no controle e monitoração dos elementos de uma rede, para atingir uma melhor utilização dos recursos e assegurar um certo nível de serviço.
		\item A motivação é estudar a eficiência no armazenamento e pesquisa de informações gerenciais, em estruturas de dados diferentes.
	\end{itemize}
\end{frame}
