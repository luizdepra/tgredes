\chapter*{Resumo}

Os sistemas de gerenciamento de redes têm como função principal o monitoramento e controle dos dispositivos que compõe as redes. Atualmente, com a rápida evolução de aplicações distribuídas, a gerência eficiente de recursos computacionais em uma rede tornou-se essencial. Destacam-se as aplicações que envolvem computação remota e todas aquelas que apresentam requisitos de nível de serviço que devem ser garantidos. O monitoramento contínuo de múltiplas características de recursos e serviços resulta em quantidades gigantescas de dados. A eficiência da gerência está diretamente ligada à forma de armazenamento e pesquisa desses dados. Este trabalho tem como objetivo a comparação experimental de duas estratégias distintas para a manipulação desses dados. O primeiro método consiste na utilização de estruturas de dados multidimensionais, em particular são utilizadas as árvores \textit{K-D} para monitoramento \textit{online} e árvores \textit{K-D-B} para monitoramento \textit{offline}. O segundo método consiste no uso de um meio tradicional, através de um banco de dados relacional, o MySQL. Os experimentos realizados foram árvore \textit{K-D} vs MySQL e árvore \textit{K-D-B} vs MySQL para busca e inserção.
