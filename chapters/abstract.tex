\chapter*{Resumo}

O gerenciamento de redes foi inicialmente impulsionado pela necessidade de monitoramento e controle dos dispositivos que compõe as redes. Atualmente, com a rápida evolução de aplicações distribuídas, o controle eficiente de recursos computacionais em uma rede tornou-se essencial. Essa eficiência está diretamente ligada à forma de armazenamento e pesquisa das informações relativas ao monitoramento. Este trabalho tem como objetivo a comparação experimental de duas maneiras distintas para tratar o problema de tratamento de dados gerenciais. O primeiro método consiste na utilização de estruturas de dados 'multidimensionais' que são adequadas a esse tipo de dado. No caso, são utilizadas as árvores \textit{K-D} para monitoramento \textit{online} e árvores \textit{K-D-B} para monitoramento \textit{offline}. O segundo método consiste no uso de um meio tradicional, através do MySQL, um SGBD muito popular. Através dos resultados obtidos experimentalmente, existe um ganho positivo com a utilização de estruturas próprias para o tratamento de dados de gerencia de redes.
