\chapter{Introdução}

  Definimos gerenciamento de rede \cite{NetworkManagement} como o processo de controle de uma complexa rede de dados visando maximizar sua eficiência e produtividade. O gerenciamento de redes têm sido um fator crítico para as organizações devido ao crescimento da complexidade dos sistemas. 
  
  O armazenamento dos parâmetros de monitoração pode ser feito de várias meios. Algumas formas viáveis de armazenamento consistem em utilizar bancos de dados relacionais ou árvores multidimensionais. O principal diferencial entre essas duas abordagens são a forma em que são feitas as pesquisas e inserções de dados. Em bancos de dados relacionais a inserção pode ser feita usando índices, que melhoram o tempo de busca mas pioram o de inserção. A busca depende da estrutura que o banco de dados e do tipo de pesquisas que serão feitas. Árvores \textit{B+} ou funções Hash são as estruturas mais comuns. Em árvores multidimensionais os registros são organizados de acordo com o valor de cada parâmetro e nível da árvore. Tanto a busca quanto a inserção ocorrem de forma semelhante, percorrendo a estrutura.
  
  O objetivo deste trabalho é comparar o desempenho do uso de árvores multidimensionais com o uso de bancos de dados relacionais, em especial o MySQL, no armazenamento de dados obtidos a partir da monitoração de um rede. Para isso, foram realizados experimentos de busca e inserção em árvore \textit{K-D}, árvore \textit{K-D-B} e banco de dados MySQL.

  Este trabalho está organizado da seguinte forma. O capítulo 2 apresenta a pesquisa multidimensional incluindo definições básicas e as estruturas de dados utilizadas no trabalho. O capítulo 3 descreve os experimentos realizados, e resultados obtidos. O trabalho finaliza no capítulo 4 com a conclusão.
