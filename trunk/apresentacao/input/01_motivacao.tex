\section{Motivação}

\begin{frame}
	\frametitle{Porque Comparar o MySQL com Árvores Multidimensionais?}
	\begin{itemize}
		\item Com a evolução das aplicações distribuídas, a gerência eficiente dos recursos computacionais tornou-se essencial.
		\item A gerência de redes consiste no controle e monitoração dos elementos de uma rede, para atingir uma melhor utilização dos recursos e assegurar um certo nível de serviço.
		\item A motivação é estudar a eficiência no armazenamento e pesquisa de informações gerenciais, usando abordagens diferentes.
	\end{itemize}
\end{frame}
