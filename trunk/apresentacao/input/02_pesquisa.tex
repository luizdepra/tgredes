\section{Pesquisa Multidimensional}

\begin{frame}
	\frametitle{Definições preliminares}
	\begin{itemize}
		\item O objetivo da pesquisa é recuperar informação a partir de uma grande massa de dados.
		\item Esses dados são divididos em registros, e cada registro possui uma chave. Essa chave é usada na pesquisa para encontrar o registro.
	\end{itemize}
\end{frame}

\begin{frame}
	\frametitle{O que é pesquisa multidimensional?}
	\begin{itemize}
		\item Um algoritmo de pesquisa recebe uma chave de pesquisa como entrada, e retorna os registros com chaves iguais à chave de pesquisa. 
		\item Quando múltiplas chaves são usadas, a pesquisa é dita multidimensional. 
		\item A pesquisa multidimensional pode ser do tipo exata, parcial ou dentro de um intervalo.
	\end{itemize}
\end{frame}

\begin{frame}
	\frametitle{Estruturas de dados multidimensionais}
	\begin{itemize}
		\item Um registro multidimensional possui múltiplas chaves. 
		\item As estruturas de dados multidimensionais indexam e dão suporte a registros multidimensionais.
		\item A pesquisa multidimensional é eficiente - logarítmica - se feita com estruturas de dados multidimensionais.
	\end{itemize}
\end{frame}
