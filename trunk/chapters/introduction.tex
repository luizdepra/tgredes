\chapter{Introdução}

  Definimos gerenciamento de rede como o processo de controle de uma complexa rede de dados visando maximizar sua eficiência e produtividade. Por isso o gerenciamento de redes têm sido um fator crítico para as organizações devido ao crescimento da complexidade dos sistemas. A monitoração concorrente de múltiplos parâmetros que representam o nível de serviço do sistema ...
  
  O armazenamento dos parâmetros de monitoração pode ser feito de varias maneiras. Algumas formas viáveis de armazenamento consistem em utilizar bancos de dados relacionais árvores multidimensionais. O principal diferencial entre essas duas abordagens são a forma em que são feitas as pesquisas e inserções de dados. Em bancos de dados relacionais a inserção pode ser feita usando índices, que melhoram o tempo de busca mas pioram o de inserção. A busca depende da estrutura que o banco de dados e do tipo de pesquisas que serão feitas. Árvores \textit{B+} ou funções Hash são as estruturas mais comuns. Em árvores multidimensionais os registros são organizados de acordo com o valor de cada parâmetro e nível da árvore. Tanto a busca quanto a inserção ocorrem de forma semelhante, percorrendo a estrutura.
  
  O objetivo deste trabalho é comparar o desempenho da inserção e busca de registros com múltiplas chaves, correspondentes a dados obtidos a partir da monitoração de uma rede. A comparação feita é entre um banco de dados relacional, o \textit{MySQL}, e duas árvores de pesquisa multidimensional, \textit{K-D} e \textit{K-D-B}.

  Este trabalho está organizado da seguinte forma. O capítulo 2 apresenta a pesquisa multidimensional com algumas definições básicas e de estruturas de dados utilizadas no trabalho. O capítulo 3 trata dos experimentos realizados mostrando os procedimentos utilizados e resultados obtidos. Finalizando com a conclusão do trabalho.
