\chapter{Introdução}

  As redes de computadores e aplicações distribuídas têm se tornado ferramentas indispensaveis para as organizações. Por esse motivo é fundamental que o serviços de rede tenham desempenho e taxa de falhas aceitaveis. Garantir a qualidade dos serviços de rede é tarefa do gerenciamento de rede \cite{NetworkManagement}, que é definido como o processo de controle de uma complexa rede de dados visando maximizar sua eficiência e produtividade.
	
	A gerencia de rede é dividida em quatro partes: elementos gerenciados, estações de gerência, protocolos de gerência e informações de gerência \cite{GerenciaRede}. Os elementos gerenciados são os componentes da redes que precisam operar adequadamente para que toda a rede ofereça os serviçoes para quais foi projetada. Cada um dos elementos gerenciados possui um software especial chamado agente, que permite que os elementos gerenciados sejam controlados e a monitorados remotamente.   O objetivo deste trabalho é comparar o desempenho do uso de árvores multidimensionais com o uso de bancos de dados relacionais, em especial o MySQL, no armazenamento de dados obtidos a partir da monitoração de um rede. Para isso, foram realizados experimentos de busca e inserção em árvore \textit{K-D}, árvore \textit{K-D-B} e banco de dados MySQL.
